# RSA CMS Admin Catalog Management — Batch 19

This batch adds admin catalog pages for Products, Categories, Brands, and Key Features.

Pages:
- `/admin/products.html`
- `/admin/categories.html`
- `/admin/brands.html`
- `/admin/key-features.html`

Scope:
- API-backed list loading
- Search and filters
- View detail drawer
- Add/edit form shell
- Product fields for visibility, package placement, price, sale price, and feature_01 through feature_10
- No image upload yet
- No delete/destructive actions
- Save buttons are intentionally disabled until the admin CRUD backend batch
